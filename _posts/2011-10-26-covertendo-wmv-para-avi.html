{EAV:2ce8babcbf73bcf7}

Hoje precisei converter um arquivo de vídeo do formato proprietário gerado pelo Windows© para um formato padrão (mp4 num container avi, por exemplo).

Para fazê-lo de forma rápida, tendo o mencoder instalado, basta executar o comando abaixo:

{% highlight bash %} $ mencoder infile.wmv -ofps 23.976 -ovc lavc \ -oac copy -o outfile.avi {% endhighlight %}
